📧 Email Notifications (EmailJS)
How to set up EmailJS (free, no backend needed):
1. Go to emailjs.com and create a free account
2. Add a service (connect your Gmail) → copy the Service ID
3. Create an email template with variables: {{to_email}}, {{subject}}, {{message}} → copy the Template ID
4. Go to Account → copy your Public Key
5. Add each employee's Gmail in their profile when creating them
6. Paste all values above and click Save
